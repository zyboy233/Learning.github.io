## python常见错误

1. SyntaxError: 'return' outside function
```javascript
while True:
    count += 1
    if count ==20:
        return
  ```
 >语法错误:return 不能在方法以外执行
 >>解决:将return放进方法体中
 
2. IndentationError: expected an indented block
```javascript
if count == 0:
print(count)
else:
  print('nothing')
```
>缩进错误:未正确缩进
>>解决:正确缩进
